* СТРАНИЦА "TRAINING START"
1) Пользователь нажимает на кнопку "Запуск программы"
1.1) Сервер добавляет новую игру
1.1) Сервер отправляет запрос координат у тренажёра
1.2) Сервер устанавливает статус для программы и для игры (далее флаг) = Ожидание отклика от тренажёра [4]
1.3) Сервер отправляет пользователю флаг
2) Пользователь опрашивает сервер на состояние флага
2.1) Если модуль обработки USART принимает координаты от тренажёра, 
	то устанавливается флаг = Расчёт позиций [6], координаты записываются в "игру"
2.2) Если тренажёр в течении какого-то количества времени (5-10 с) не вернул координаты, 
	то устанавливается флаг = Ожидание ввода положения тренажёра [5]
2.3) Сервер отправляет пользователю флаг
3) Если флаг = Ожидание ввода положения тренажёра [5], то пользователь указывает 
	в интерфейсе координаты тренажёра. 
3.1) Сервер устанавливается флаг = Расчёт позиций [6], координаты записываются в "игру"
3.2) Сервер на основе координат тренажёра расчитывает параметры выстрелов
3.3) Сервер отправляет пользователю флаг
4) Пользователь переходит на страницу "GAME START"

* СТРАНИЦА "GAME START"
5) Пользователь открывает страницу "GAME START"
5.1) Сервер загружает программу в тренажёр
5.2) Сервер устанавливает флаг = Загрузка программы
5.3) Модуль обработки USART загружает программу в тренажёр
5.4) Модуль обработки USART устанавливает флаг = Исполнение программы
5.5) Сервер отправляет пользователю флаг
7) Идёт исполнение программы

8) Если модуль обработки USART принимает команду завершения тренировки, 
	то устанавливается флаг = Программа завершена
9) Если пользователь прерывает программу из интерфейса, 
	то устанавливается флаг = Программа остановлена пользователем
9.1) Сервер отправляет тренажёру команду завершения программы
10) Если пользователь прерывает программу иным образом,
	то модуль обработки USART устанавливает флаг = Программа остановлена пользователем
11) Если модуль обработки USART принимает команду повтора программы,
	то игра клонируется, для клона устанавливается флаг = Исполнение программы

Статусы:
0 = Остановлена
1 = Программа остановлена пользователем
2 = Программа завершена
3 = Программа приостановлена
4 = Ожидание отклика от тренажёра
5 = Ожидание ввода положения тренажёра
6 = Расчёт позиций
7 = Загрузка программы
8 = Исполнение программы
